Four operating modes
☀️ Morning briefing
Auto at 8:00 am. Reads Today + This Week from Reminders. Sends Telegram message: top priority, task count, anything overdue. Writes morning Journal entry via Shortcuts.
🌙 Evening review
Auto at 8:30 pm. Sends 3 guided prompts in Telegram. Your replies are captured → writes to Journal evening entry. Flags anything in Today that wasn't completed.
⚡ Natural capture
Any time. "add call John to this week" → parsed by Claude → reminders-cli adds it to the right list. Confirms back in Telegram. "what's on today?" → reads and replies.
🗓 Weekly review
Auto Sunday 7 pm. Guided conversation in Telegram: brain dump → look back → groom Reminders (moves stale items) → look ahead → creates new scratch pad .md for the week.
Build decisions — what to adapt vs what to add
Keep from PyClaw
Telegram bot handler, Claude API calls, file read/write for .md, conversation state management
Add new
reminders-cli wrapper tool, Shortcuts bridge for Journal, launchd .plist for scheduling, Apple Stack system prompt, 4 mode handlers
