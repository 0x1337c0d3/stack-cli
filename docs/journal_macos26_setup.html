Initial Mac setup
1

Open Journal — it's in Applications or search with ⌘ Space → type "Journal"

Pre-installed with macOS 26 Tahoe — no download needed

2

Follow the onscreen setup, enable Touch ID lock when prompted

3

Create your three journals — click the + button in the sidebar above All Entries, name it, then press Return

Control-click any journal to set its colour and icon

4

Set reminders: Journal menu → Settings → Journaling Habit

Morning: 8:00 am · Evening: 8:30 pm · Weekly: Sunday 7:00 pm

5

Enable sharing: Settings → Allow sharing from other apps

Lets you send articles, music, links into Journal from any app via the Share menu

Your three journals

Daily Review

Morning intention + evening review — your core daily habit. Short entries, 5 min max.

Weekly Reflection

Sunday evening brain dump, look back and look ahead. One entry per week.

Capture

Random thoughts, ideas, things shared from other apps. No structure — just dump it here.

💡 On iPad: use the Pencil to handwrite in Daily Review — it feels much more like a real journal and Apple Pencil handwriting is now native in macOS 26 Journal too.

How it all connects — daily flow
AM Journal → Daily Review — set intention, dump anything unplanned into Reminders Inbox
Day Reminders for tasks · Notes for thinking · Journal Capture for random ideas via Share menu
PM Journal → Daily Review — three wins, move anything unfinished in Reminders
Sun Journal → Weekly Reflection — brain dump, groom Reminders, create new Notes scratch pad for the week
✅ All three journals sync automatically across your Mac, iPhone, and iPad via iCloud — end-to-end encrypted, Face ID / Touch ID locked.
